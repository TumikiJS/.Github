# TumikiJS

## 🌐Language
[JA](https://github.com/TumikiJS) / [EN](https://github.com/TumikiJS/.Github/blob/main/profile/EN_README.md)

## 💡Summary
TumikiJSは、あらゆる操作をストレスなく扱うためのモジュール群です。複雑かつ理解しがたいプログラムを一貫性があるシンプルな記述で利用できます。

## ✨Features
### 理不尽な制約から解放
ブラウザや端末固有の制限を最小限の方法で自動的に解除します。
